An Introduction to Homestuck

Homestuck is a webcomic run by Andrew Hussie. Homestuck, which is part of MS Paint Adventures, is about four thirteen-year-old children who play a video game called Sburb, and then enter a realm called the Medium. The game is as old as time itself, and the object of the game is to create a new universe. Once each universe has reached maturity, the dominant species of that universe will then proceed to play the game and start their own session. This continues until a session is considered to be "null", or unbeatable. If a session is null, then a "scratch", or reset, may be implemented in order to restart the universe from the beginning and try to create a valid session. So far in the story, two species have played the game: humans and trolls.
